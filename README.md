### Prerequisites
- Node.js v16+
- npm/yarn
- Firebase CLI

### Installation
```bash
# Clone repository
git clone https://github.com/your-org/halalchain-marketplace.git

# Install dependencies
cd halalchain-marketplace

# Set up environment variables

```

### Running Locally
```bash

npm run dev

# Build for production
npm run build && npm start
```
